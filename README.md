# Skeletons - Three.js Game (Vue Powered)

A browser game built with **Three.js** and wrapped inside a **Vue.js application**.
The project focuses on lightweight rendering, stylized visuals, and mobile-friendly gameplay.

## Tech Stack

* **Three.js** – 3D rendering engine
* **Vue.js** – application wrapper and UI management
* **Vite** – fast build tool and development server
* **JavaScript (ES6+)**
* **GLB / GLTF** assets for animated 3D characters

## Features

* Real-time 3D gameplay in the browser
* Mobile and desktop support
* Optimized GLB models and animations
* Sound effects system
* Cartoon-style visual effects
* Lightweight rendering for better FPS

## Mobile Support

The game can be installed as a **Progressive Web App (PWA)** on mobile devices.

To install:

1. Open the game in your mobile browser.
2. Select **Add to Home Screen**.
3. Launch the game directly from your device home screen.
