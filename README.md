A lightweight **browser game built with Three.js and wrapped in Vue.js**.
The game focuses on **stylized visuals, optimized 3D assets, and smooth performance on mobile and desktop browsers**.

🎮 Live Demo - [https://a2020develop.github.io/skeletons](https://a2020develop.github.io/skeletons)

# ✨ Features

* 💥 Online multiplayer with **Firebase Realtime DB**
* ⚡ Fast rendering with **Three.js**
* 🧩 Modular architecture using **Vue.js**
* 📱 Mobile friendly
* 🎮 Real-time gameplay
* 🔊 Sound effects system
* 🎨 Cartoon-style visual effects
* 📦 Optimized GLB/GLTF models
* 🚀 Fast builds using **Vite**


# 📱 Install on Mobile (PWA)

You can install the game on your phone and launch it like a native app.

### Android (Chrome)

1. Open the game link.
2. Tap **⋮ menu**
3. Tap **Add to Home Screen**
4. Launch the game from your home screen.

### iOS (Safari)

1. Open the game in Safari.
2. Tap the **Share button**
3. Select **Add to Home Screen**
4. Launch the game from your home screen.


# ⚙️ Performance Notes

To maintain high FPS:

* Models are optimized GLB files
* Minimal post-processing
* Efficient texture usage
* Mobile-friendly shaders
* Basic Material usage with no scene light calculations
